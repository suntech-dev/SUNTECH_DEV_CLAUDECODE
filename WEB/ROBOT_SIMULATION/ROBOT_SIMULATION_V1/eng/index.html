SunTech Co-Robot Sewing Simulation User Manual

1. Introduction

This program simulates the workflow of a Co-Robot working together with Computer Sewing Machines.

Target Product: Designed based on SunTech ST-ES612, ST-EA612 Robot models.

2. Getting Started - Mode Selection

When you start the program, a screen to select the number of sewing machines appears.

Mode Description Layout
2-Machine Mode Operate 2 sewing machines Left(M1) + Right(M2)
3-Machine Mode Operate 3 sewing machines Left(M1) + Top(M2) + Right(M3)

Tip: You can switch modes anytime using the [Change Mode] button in the header.

3. Basic Settings

Item Description Default
Workers Number of workers operating simultaneously. Used for PPH (Productivity Per Hour) calculation. 2
Pallet Prep Time Time for a worker to prepare new fabric (Pallet) on the material table. 58 sec
Pallet Count Total number of pallets circulating in the system (3-7). More pallets than machines allows advance material preparation, improving utilization. 5-6
Pallet Count:
• More pallets reduce material waiting time, improving productivity.

4. Work Time Settings

4-1. Robot Speed Selection

Set the robot's operating speed using presets or manual input.

Mode Description
Manual Freely enter time values for each machine (default)
75% ~ 100% Selecting a speed preset automatically sets Initial Load, Unload & Load, and Return Time
Note: Sewing Time is independent of robot speed and can always be manually modified.

Reference values by speed (75% / 85% / 100%):

Machine Item 75% 85% 100%
Machine 1 Initial Load 12.5s 11s 8.5s
Unload & Load 17.5s 15.5s 13s
Return 10.5s 9s 7.5s
Machine 2 Initial Load 13.5s 11.8s 9s
Unload & Load 18.5s 16.3s 13.5s
Return 11.5s 9.8s 8s
Machine 3 Initial Load 12.5s 11s 8.5s
Unload & Load 17.5s 15.5s 13s
Return 10.5s 9s 7.5s

Tip: Intermediate values (80%, 90%, 95%) are automatically calculated by linear interpolation between 75-85 and 85-100 ranges.

4-2. Time Item Description

Set 4 time values for each machine (in seconds):

Item Description
Initial Load Time Time for the robot to initially load a pallet onto the sewing machine
Unload & Load Time Time to unload completed pallet + load new pallet using 360-degree rotation (from 2nd cycle onwards)
Return Time Time for the robot to return to its home position after retrieving the completed pallet
Sewing Time Time for the sewing machine to complete the sewing operation
Note: Longer sewing time gives the robot more time to service other machines. Shorter sewing time makes the robot busier, increasing machine wait time.

5. Simulation Duration

Set the total time for the simulation to run.

  • Can be set from minimum 1 hour to maximum 24 hours
  • Hours and minutes can be selected separately
  • When 24 hours is selected, only 00 minutes is available

6. Running the Simulation

There are two ways to run:

Quick Calc

  • Calculates results instantly
  • Results only, no animation
  • Use when you want quick results

Realtime Calc

  • Visually observe robot movements
  • Adjustable speed (0.1x ~ 100x log scale)
  • Timeline slider for rewind/skip
  • Pallet carrying state visualization
  • Pause / Resume available
  • Use to understand the process

Realtime Calc Mode Details

Feature Description
Realtime Stats Bar Displays production, wait time, robot active/idle time for each machine in real-time during simulation
Robot Movement Animation Visually shows the robot's path and elapsed time between table and machines
Machine Progress Bar Shows sewing progress for each machine as a progress bar
Status Display Shows current status (Sewing, Waiting, Moving, etc.) as text for robot and machines
Speed Fine Control A log-scale slider allows fine-grained speed adjustment from 0.1x to 100x. Preset buttons (0.1x, 0.5x, 1x, 5x, 10x, 50x, 100x) are also available
Pallet Visualization Shows the robot's carrying state during movement. A blue P badge appears when carrying a new pallet, and a gold ✓ badge when carrying a completed item
Timeline Slider Displays simulation progress as a slider. Drag to jump to any point in time. Use -10s/+10s buttons for quick navigation

Event Log

All events are recorded in a table format during simulation.

  • Time(sec): Event occurrence time
  • Event Type: Depart, Loading, Sewing Done, Production, etc.
  • Machine: Target machine for the event
  • [Collapse/Expand] button to collapse or expand long logs

7. Interpreting Results

7-1. Per-Machine Results

Item Meaning
Total Production Number of completed products from each machine
Total Wait Time Time waited for robot after sewing completion (shorter is better)

7-2. Overall Metrics

Item Meaning
Total Production Sum of completed products from all machines
Robot Utilization Percentage of time the robot was actually working (higher = better utilization)
Robot Active Time Total time the robot spent on movement, loading, unloading, etc.
Simulation Time Total simulation time configured
PPH Person Per Hour - Number of products per worker per hour
Formula: Total Production ÷ Hours ÷ Workers
Machine Utilization Average sewing utilization across all machines (higher = less idle time)
Machine Active Time Total actual sewing time across all machines

7-3. Efficiency Improvement

If simulation results show high wait time or low utilization, consider the following:

Improvement Effect When to Apply
Reduce Pallet Prep Time Fewer Unload-only occurrences, reduced machine wait time When prep time is close to sewing time
Increase Pallet Count Less material waiting, continuous operation possible When robot frequently enters "Pallet Wait" state
Increase Robot Speed Shorter service interval between machines When robot utilization is low and machine wait time is high
💡 Understanding Unload-only

What is Unload-only? When the robot retrieves a completed product but no new pallet is ready, it only unloads the product and leaves the machine EMPTY.

Cause: When Pallet Prep Time (e.g., 58s) is close to Sewing Time (e.g., 63s), new material may not be ready when the robot brings back the completed product.

Impact: Frequent Unload-only events cause machines to stay EMPTY longer, dramatically increasing total wait time.

Solutions:
  • Reduce Pallet Prep Time to speed up material supply
  • Increase Pallet Count to ensure spare materials
  • [Alternative] Increase workers to parallelize pallet preparation

8. Result Charts

After running a simulation, 4 charts are automatically displayed at the bottom of the results area.

Chart Type Description
Production by Machine Bar Chart Compares completed quantity per machine. Easily identify production variance between machines.
Wait Time by Machine Bar Chart Compares total time each machine waited for the robot. Helps identify bottlenecks.
Robot Time Usage Doughnut Chart Shows Active Time vs Idle Time ratio for the robot. Utilization(%) is displayed in the center.
Machine Time Usage Doughnut Chart Shows Sewing Time vs Idle Time ratio for all machines. Utilization(%) is displayed in the center.
Tip: If a specific machine shows exceptionally long wait time in the bar chart, its sewing time may be shorter or it may have lower priority in the robot's travel path.

9. Scenario Save & Comparison

Run simulations with different settings, save the results, and compare them side by side.

9-1. Saving Scenarios

Item Description
How to Save Click the [Save Scenario] button in the results area after running a simulation
Scenario Name Enter a name when saving (e.g., "Speed85% 3machines", "7pallets test")
Max Scenarios Up to 5 scenarios can be saved
Saved Data Basic settings, input time values, and simulation results are all saved together

9-2. Comparing Scenarios

When 2 or more scenarios are saved, the comparison section is automatically displayed.

Comparison Item Description
Performance Table Compares key metrics (Total Production, PPH, Utilization, etc.) side by side. Best values are highlighted in green.
Total Production Compares total production across scenarios in a bar chart
PPH Comparison Compares productivity per worker per hour across scenarios in a bar chart
Efficiency (Radar) Compares robot utilization, machine utilization, etc. at a glance using a radar chart

9-3. Managing Scenarios

  • Delete individual scenarios using the × button on each scenario item
  • Use the [Delete All] button to remove all saved scenarios at once
  • Each scenario is color-coded in the saved list for easy identification
Usage Example: Run simulations at robot speeds of 75%, 85%, and 100%, save each as a scenario, and instantly compare productivity differences across speed settings.

10. PDF Export

Simulation results can be saved as a PDF file.

  • After running a simulation, the [Save PDF] button appears in the results area
  • Filename is automatically generated as CoRobot_[mode]_Report_date_time.pdf

PDF Contents:

Section Content
Basic Settings Workers, Pallet Prep Time, Pallet Count
Input Values Initial Load, Unload & Load, Return, Sewing Time per machine
Simulation Results Production, theoretical max, wait time, efficiency(%) per machine
Summary Total Production, PPH, Robot/Machine Utilization, Active Time
Event Log Full simulation event records (Time, Event, Machine, Description)
