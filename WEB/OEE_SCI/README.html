1 프로젝트 개요

OEE_SCI는 인도네시아 縫製(봉제) 공장의 IoT 재봉기 장비를 실시간으로 모니터링하는 OEE(Overall Equipment Effectiveness) 관리 시스템입니다. MAC 주소 기반으로 장비를 자동 등록하고, SSE를 통해 실시간 데이터를 스트리밍합니다.

OEE 공식

5 API 구조

응답 형식

// 성공
{"code": "00", "msg": "success", "data": {...}}

// 실패
{"code": "99", "msg": "error message"}

IoT 장비 데이터 수신 (POST api/iot/receive.php)

// Request Body
{
  "mac": "AA:BB:CC:DD:EE:FF",
  "count": 100,
  "status": 1
}

// Response
{
  "code": "00",
  "msg": "success",
  "machine_id": 5,
  "oee": {"availability": 95, "performance": 88, "quality": 99}
}

SSE 스트리밍 (GET api/iot/sse.php)

// PHP SSE 헤더
header('Content-Type: text/event-stream');
header('Cache-Control: no-cache');
header('Connection: keep-alive');

// 클라이언트 연결 (JavaScript)
const es = new EventSource('/api/iot/sse.php?factory_id=1&line_id=2');
es.onmessage = (e) => {
  const data = JSON.parse(e.data);
  // 실시간 OEE 업데이트
};
