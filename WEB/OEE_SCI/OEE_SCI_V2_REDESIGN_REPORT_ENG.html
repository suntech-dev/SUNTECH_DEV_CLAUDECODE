AI Intelligence
New AI Features — Detailed

Real-time AI analytics powered by a self-implemented statistics engine (Exponential Smoothing, Z-Score, Linear Regression) — no external AI API required for core analysis.

NEW AI Intelligence Dashboard — Unified AI Command Center

A dedicated dashboard integrating 5 AI analysis sections in a single screen. Auto-refreshes every 60 seconds, with cascading Factory→Line→Machine filters and date range selection. Signage-optimized version v5 (1920×1080) available.

  • OEE Forecast summary card (next 4-hour prediction + trend direction badge)
  • Real-time display of at-risk machine count and active anomaly count
  • Line-level AI Health Index (progress bar + Normal/Caution/Danger grade)
  • Predictive Maintenance recommendation list (sorted by risk score)
  • Real-time AI event streaming feed (slide-in animation, up to 15 events)
  • Production Optimization suggestion cards (P1/P2/P3 priority)

OEE Prediction (Predictive OEE)

Learns hourly OEE patterns from the past 30 days to forecast the next 4–8 hours. Displays a 90% Confidence Interval (upper/lower bounds).

Exp. Smoothing + Day×Hour Seasonality
🚨

Anomaly Detection

Monitors OEE, Availability, and Quality per machine in real time. Triggers instantly when values exceed ±2σ or when cascading anomalies appear on the same line.

Z-Score + Cascading Detection
🔧

Predictive Maintenance

Calculates a maintenance risk score (0–100) using Runtime (40%) + Downtime Frequency Growth Rate (35%) + OEE Instability (25%).

Weighted Scoring (3 Factors)
📊

AI Quality Sentinel

Automatically computes Pareto analysis (80% rule), 24-hour defect heatmap, machine risk ranking (last 7d vs prior 7d growth rate), and Pearson OEE correlation.

Pareto + Heatmap + Pearson
⚡

Real-time AI Streaming

Streams OEE anomalies, new downtime events, and maintenance risk alerts every 15 seconds via SSE. Up to 15 slide-in event cards displayed.

SSE + 5-min Session + Auto-reconnect

Production Optimization

Identifies Availability / Performance / Quality bottlenecks per line, estimates potential OEE gain, and auto-generates 4 improvement suggestions with P1/P2/P3 priority.
