280CTP_IoT_INTEGRATED_V1 를 가지고 UART 통신 테스트만을 위한 버전 만들기
- Wifi 기능 사용 안함.
- LCD 에 표현하는 기존 메뉴 모두 사용 안함.
- 오직 데이터 통신이 잘 되고 있는지 확인 용도의 프로젝트이다.
- 다른 MCU로부터 UART 통신으로 데이터를 받으면
- 280CTP_IoT_UART_TEST_V1의 LCD 화면에 데이터를 보여준다.
- 데이터가 하나의 화면을 벗어나면 Up, Down 버튼을 이용해서 확인을 한다.
- 280CTP_IoT_UART_TEST_V1의 USB port 와 연결한 PC 에서도 데이터를 볼 수 있어야 한다.
